## Prerequisites

- SDL 2.0, SDL_image 2.0, SDL_ttf 2.0, and SDL_mixer 2.0 are required.
- `font.ttf` must be placed in `./src/resources`. **[Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P)** is recommended.
- The following files must be in `./src/resources`:
  - `bgm.ogg`
  - `se_move.ogg`
  - `se_rotation.ogg`
  - `se_shock.ogg`
